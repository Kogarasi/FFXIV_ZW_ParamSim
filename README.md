# FFXIV キャプションメイカー

リンク先:  
http://kogarasi.github.io/FFXIV_CaptionMaker/

## 使い方

ページ左側のパラメータをいじることで、以下の項目が調整可能です。

* キャプション内容
* キャプション文字色
* キャプション文字サイズ
* キャプション使用フォント（ただし、表示しているブラウザで利用可能な物に限る）
* キャプション位置

パラメータをいじったあとに、画像をローカルへ保存（できるはず）

作成した画像はご自由にお使いください。

## 推奨環境

Windows or Mac OS / Chrome
